# RP-Bot-Project

A powerful **Node.js + MongoDB powered Roleplay Discord bot** designed for serious roleplay communities such as **ERLC (Emergency Response: Liberty County)** and other roleplay servers.

This project is actively being improved and updated over time with new features, systems, and optimizations.

---

# Features

- Discord.js powered bot
- MongoDB database integration
- Roleplay management systems
- ERLC community support
- Scalable command structure
- AI integration support using Google Gemini
- More features coming soon

---

# Installation Guide

## Requirements

Before installing the bot, make sure you have:

- **Node.js** installed on your computer
- **Visual Studio Code** (recommended) or any code editor
- A Discord Bot Token
- A MongoDB database connection

---

# Step 1: Install Node.js

Download and install Node.js from the official website:

https://nodejs.org/

After installation, check if Node.js is installed:
bash
node -v 

--

Step 2:
 Download The Project
Download this repository from GitHub.
Extract the files.
Open the project folder inside Visual Studio Code.

--
Step 3: Install Dependencies
Open the terminal in Visual Studio Code:
Click Terminal from the top menu.
Select New Terminal.
Run this command:
npm install discord.js @google/genai mongoose
This will install the required packages:

discord.js → Discord bot framework
@google/genai → Google Gemini AI integration
mongoose → MongoDB database connection
--
now start your bot
```node index.js ```
then Type in new terminal
Node Deploy.js

⚠️ Please note the following:
This process will host your bot locally on vs code any time you close vs code the bot will be offline. to make it permanent use dedicated Hosting provider like a VPS servers like Azure oracle or anything like that. For easy free hosting Choose WispByte katabump or Render or railway.
